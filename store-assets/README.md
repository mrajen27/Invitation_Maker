# Store assets

## Play Store icon (512×512)

Upload `play_store_icon_512.png` in [Google Play Console](https://play.google.com/console) → **Store presence** → **Main store listing** → **App icon**.

The on-device launcher uses the adaptive icon in `app/src/main/res/drawable/ic_launcher_*.xml` (maroon gradient, cream invitation envelope, gold seal, toran).
