# Store assets

## Play Store icon (512×512)

Upload `play_store_icon_512.png` in [Google Play Console](https://play.google.com/console) → **Store presence** → **Main store listing** → **App icon**.

The on-device launcher uses the adaptive icon in `app/src/main/res/drawable/ic_launcher_*.xml` (maroon gradient, cream invitation envelope, gold seal, toran).

## Feature graphic (1024×500)

Upload `feature_graphic_1024x500.png` in Play Console → **Main store listing** → **Feature graphic**.

| Spec | Value |
|------|--------|
| Size | 1024 × 500 px |
| Format | PNG |
| Max size | 15 MB (file is ~800 KB) |

## Phone screenshots (9:16)

Upload **2–8** images from `phone-screenshots/` in Play Console → **Main store listing** → **Phone screenshots**.

| Spec | Value |
|------|--------|
| Aspect ratio | **9:16** (portrait) |
| Size used | **1080 × 1920** px |
| Format | PNG |
| Max per file | 8 MB |

| # | File | Screen |
|---|------|--------|
| 1 | `01_home_categories.png` | Home — event categories |
| 2 | `02_wedding_templates.png` | Wedding template picker |
| 3 | `03_editor_english.png` | Editor (English) |
| 4 | `04_editor_tamil.png` | Editor (Tamil) |
| 5 | `05_invitation_preview.png` | Generated invitation |
| 6 | `06_save_and_whatsapp.png` | Save & WhatsApp share |

You may upload all 6, or pick any **4** minimum. Order above is recommended for the store carousel.
